### Cox 风险比例模型

------

[TOC]

##### 01：概述

- 是一种用于分析“事件发生时间”（Time-to-Event）的**半参数回归模型**，它在**不假设基线风险函数具体形式**的前提下，评估**多个协变量（特征）对事件发生风险的相对影响**。

##### 02：非参数模型

| 类型                          | 特点                                                         | 举例                                                         |
| ----------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 参数模型（Parametric）        | 假设数据服从某种**已知分布形式**，只需估计有限个参数         | 线性回归（假设误差正态）、Weibull 生存模型（假设风险函数为 Weibull 分布） |
| 非参数模型（Non-parametric）  | **不做分布假设，直接从数据学习函数形状**，参数数量随数据增长 | Kaplan-Meier 生存曲线、核密度估计                            |
| 半参数模型（Semi-parametric） | 一部分用参数形式，另一部分保持非参数                         | **Cox 模型**                                                 |

##### 03：Cox 模型核心公式

$$
h(t \mid \mathbf{x}) = h_0(t) \cdot \exp(\beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p)
$$

其中：

- $ h(t \mid \mathbf{x}) $：给定特征 $\mathbf{x}$ 时，客户在时间 $t$ 的瞬时失活风险
- $ h_0(t) $：基线风险函数，所有特征为0时的风险；
- $ \beta_j $：第 $j$ 个特征的系数，回归系数；
- $ x_j $：第 $j$ 个特征的取值（如 last_7d_spend）
  - 对于 last_7d_spend：β = -0.8 ：它出现在指数项中：$\exp(-0.8 \times \text{last\_7d\_spend})$，HR = 0.45;
  - 表示：**last_7d_spend 每增加1单位，失活风险降低55%风险降低了（ 1 − 0.45 = 55%）**。
- $ \exp(\boldsymbol{\beta}^\top \mathbf{x}) $：**风险比（Hazard Ratio, HR）**：个体相对于基线的风险倍数。

##### 04：基线风险函数（Baseline Hazard Function）

- 是当所有协变量 $x=0 $ 时的风险函数，即“基准个体”在时间 *t* 的瞬时失活率。不一定是真实的客户，是一个参考点。
- 含义：整个客户群体随时间变化的“基础流失节奏”，剥离了个体差异后的共性模式。而每个具体客户的风险 = 这个**共性节奏** × **他自身特征带来的放大/缩小倍数**（即：$ \exp(\boldsymbol{\beta}^\top \mathbf{x}) $）。
- **非参数部分**：数据不服从某种分布，没有预设形式，其“形状”完全由数据驱动，在理论上需要无限维参数才能描述。

##### 05：删失（Censoring）

- 删失表示我们**无法观察到某个个体的确切事件发生时间**（如死亡、复发等），但知道该事件**在其观测结束前发生未发生**。

- | 类型     | 含义                | 典型场景                     |
  | -------- | ------------------- | ---------------------------- |
  | 右删失   | 事件时间 > 观测时间 | 随访结束时事件未发生         |
  | 左删失   | 事件时间 < 观测时间 | 事件发生在首次观测之前       |
  | 区间删失 | 事件时间 ∈ (L, R]   | 定期检查中事件发生在两次之间 |

###### 公式

数学表示：观测值为 $ (t_i, \delta_i) $，其中
$$
\delta_i =
\begin{cases}
1, & \text{事件发生（未删失）} \\
0, & \text{右删失}
\end{cases}
$$

##### 07：模型评估

- **exp(coef)** **= 风险比（Hazard Ratio, HR）**

  - **HR < 1** → ，$ \beta$ 是负数，该特征值越高，**失活风险越低**（保护因素 ✅）

  - **HR > 1** → ，$ \beta$ 是正数，该特征值越高，**失活风险越高**（危险因素 ❌）

    - HR 是指数函数，不能直接比较强度，进行对数转化。

      | **真实效应** | **HR 值** | 距离 HR=1 的“差距” | **统一比值维度（做对数转换）** |
      | ------------ | --------- | ------------------ | ------------------------------ |
      | 风险翻倍     | 2.0       | 2.0 − 1 = **1.0**  | log(2) ≈ +0.693                |
      | 风险减半     | 0.5       | 1 − 0.5 = **0.5**  | log(0.5) ≈ −0.693              |

- **C-index** ：一致性指数：衡量模型预测的**高风险的人是否真的比低风险的人更早失活。**

  - 示例：实际失活概率 A > B > C，那么 C-index = 1.0（完美）。

    | **客户** | **预测风险得分（越高越危险）** | **实际是否失活（event）** | **失活时间（天）** |
    | -------- | ------------------------------ | ------------------------- | ------------------ |
    | A        | 0.9                            | 1（失活）                 | 第 10 天           |
    | B        | 0.6                            | 1（失活）                 | 第 25 天           |
    | C        | 0.2                            | 0（未失活）               | —（删失）          |

- **p 值**：用来判断一个特征是否“**显著影响****”的关键指标**。【只看有没有影响，影响强度看 HR】

  - **p < 0.05：**有足够证据**拒绝 H₀** → 特征**显著影响**失活风险 ✅
  - **p ≥ 0.05：**没有足够证据拒绝 H₀ → 不能确定该特征是否有真实影响 ❓
