##### 01：多层感知机（Multilayer Perceptron, MLP）

- 是最基础、最重要的前馈人工神经网络（ANN） 模型之一，广泛用于分类和回归任务。
- MLP 由三层（或更多）全连接层组成；

##### 02：MLP 基本结构

| 层                            | 作用               | 节点数                             |
| :---------------------------- | :----------------- | :--------------------------------- |
| **输入层（Input Layer）**     | 接收原始特征       | = 输入特征维度（如 784 for MNIST） |
| **隐藏层（Hidden Layer(s)）** | 提取非线性特征表示 | 可设多个，每层节点数可调           |
| **输出层（Output Layer）**    | 输出预测结果       | = 类别数（分类）或 1（回归）       |

###### 关键特性

- **全连接（Fully Connected）**：每一层的每个神经元都与下一层所有神经元相连。
- **前馈（Feedforward）**：信息单向流动（输入 → 隐藏 → 输出），无循环或反馈。

##### 03：隐藏层（H Layers）

- 没有隐藏层，神经网络只能学习线性关系；有了隐藏层（**非线性激活函数**），它才能**拟合复杂的非线性曲线**。

###### 公式推导

$$
H = f_1\left( \mathbf{W}_1 \mathbf{x} + \mathbf{b}_1 \right)\\
\mathbf{O} = f_2\left( \mathbf{W}_2 \cdot \underbrace{f_1\left( \mathbf{W}_1 \mathbf{x} + \mathbf{b}_1 \right)}_{\text{非线性变换}} + \mathbf{b}_2 \right)
$$

- 矩阵 $x$ 表示 n 个样本的，其中每个样本具有 d 个输入特征；
- 激活函数：$f_1$ 是对每个隐藏单元应用非线性的激活函数，激活函数的输出被称为活性值。

###### 作用

- 通过非线性激活函数，将输入映射到一个更高维特征空间，从而使模型能够学习复杂的、非线性的模式。
- 没有隐藏层，就等于直接用原始特征做预测，无法捕捉特征之间的相互作用。有了隐藏层，可以自动学习抽象特征表示及特征间的相互作用。

###### 通用近似原理

- 只要隐藏层足够宽（更多神经元），哪怕只有一层，MLP 就可以以任意精度逼近任何连续函数。

##### 04：激活函数（activation function）

- 神经网络中每个神经元的核心组件，它的作用是**对输入信号进行非线性变换**，计算加权和并加上偏置来确定神经元是否应该被激活。
- 加入非线性激活函数，神经网络可以**拟合任意复杂的非线性函数**（通用逼近定理）。
- **偏置**：每个神经元的一个可学习参数，用于**在激活函数之前对加权输入进行平移**，使模型能够拟合更复杂的数据分布。

###### 常用的激活函数

- 结合非线性函数来构建具有**更强表达能力的多层神经网络架构**。
  - ReLU 函数
  - Sigmoid 函数

##### 05：前向传播（Forward Propagation）

- 将输入数据**从输入层逐层传递到输出层**，计算网络最终预测结果的过程。

##### 06：反向传播（backward propagation）

- 计算神经网络**参数梯度的方法**。
- 利用**链式法则（Chain Rule）**，按相反的顺序**从输出层到输入层遍历网络**。 该算法存储了计算某些参数梯度时所需的任何中间变量（偏导数），用于更新参数。

###### 两者的区别

- 在训练神经网络时，在初始化模型参数后**， 交替使用前向传播和反向传播，利用反向传播给出的梯度来更新模型参数**。

| 特性             | 前向传播            | 反向传播                  |
| :--------------- | :------------------ | :------------------------ |
| **方向**         | 输入 → 输出         | 输出 → 输入               |
| **目的**         | 生成预测            | 计算梯度                  |
| **依赖**         | 当前参数（W, b）    | 损失函数 + 链式法则       |
| **计算内容**     | 激活值 a(l)*a*(*l*) | 梯度 ∂L∂W(l)∂*W*(*l*)∂*L* |
| **是否更新参数** | ❌ 否                | ✅ 是（配合优化器）        |
| **计算量**       | 较小                | 较大（但可自动微分）      |

##### 07：块（Block）

- 一种比“层（Layer）”更高层次的模块化设计，由**多个层按特定方式组合而成**，形成一个**功能完整、可复用的子网络结构**。

###### 自定义块

- 它包含一个多层感知机，其具有256个隐藏单元的隐藏层和一个10维输出层。

```python
class MLP(nn.Module):
    # 用模型参数声明层。这里，我们声明两个全连接的层
    def __init__(self):
        # 调用MLP的父类Module的构造函数来执行必要的初始化。
        # 这样，在类实例化时也可以指定其他函数参数，例如模型参数params（稍后将介绍）
        super().__init__()
        self.hidden = nn.Linear(20, 256)  # 隐藏层
        self.out = nn.Linear(256, 10)  # 输出层

    # 定义模型的前向传播，即如何根据输入X返回所需的模型输出
    def forward(self, X):
        # 注意，这里我们使用ReLU的函数版本，其在nn.functional模块中定义。
        return self.out(F.relu(self.hidden(X)))
```

###### 在前向传播函数（forward）中执行代码

- 定义：**输入如何一步步变成输出**。

- 有时我们希望合并既不是上一层的结果也不是可更新参数的项， 我们称之为*常数参数*（constant parameter）

  ```python
  class FixedHiddenMLP(nn.Module):
      def __init__(self):
          super().__init__()
          # 不计算梯度的随机权重参数。因此其在训练期间保持不变
          self.rand_weight = torch.rand((20, 20), requires_grad=False)
          self.linear = nn.Linear(20, 20)
  
      def forward(self, X):
          X = self.linear(X)
          # 使用创建的常量参数以及relu和mm函数
          X = F.relu(torch.mm(X, self.rand_weight) + 1)
          # 复用全连接层。这相当于两个全连接层共享参数
          X = self.linear(X)
          # 控制流
          while X.abs().sum() > 1:
              X /= 2
          return X.sum()
  ```

##### 08：参数管理

######  参数访问

- 通过索引来访问模型的任意层参数；
- 参数是复合的对象，包含值、梯度和额外信息；

###### 参数共享

- 在多个层间共享参数： 我们可以定义一个**稠密层**，然后使用它的参数来设置另一个层的参数。

```python
# 我们需要给共享层一个名称，以便可以引用它的参数
shared = nn.Linear(8, 8)
net = nn.Sequential(
  nn.Linear(4, 8),     # 第 0 层：带参数的“仿射变换”层 
  nn.ReLU(),					 # 第 1 层：无参数的“激活函数”层
  shared,
  nn.ReLU(),
  shared,
  nn.ReLU(),
  nn.Linear(8, 1))     # 第 6 层
net(X)
# 检查参数是否相同
print(net[2].weight.data[0] == net[4].weight.data[0])
# 修改第 2 层权重
net[2].weight.data[0, 0] = 100
# 确保它们实际上是同一个对象，而不只是有相同的值
print(net[2].weight.data[0] == net[4].weight.data[0])
```

- 总共有 7 个模块，但不一定有 7 层；
- 神经网络有多少层”**通常指的是有多少个带参数的仿射变换层**（如 Linear、Conv）。
  1. `nn.Linear(4, 8)` → **第1个权重层，不是输入层**
  2. `shared`（第一次出现）→ **第2个权重层**
  3. `shared`（第二次出现）→ **仍然是同一个权重层！**（参数共享，不算新层）
  4. `nn.Linear(8, 1)` → **第3个权重层，该网络为三层MLP**

###### 延迟初始化

- 直到数据第一次通过模型传递时，框架才会**动态地推断出每个层的大小**。

##### 09：自定义层

- 标准层无法满足需求，需要自己灵活的设计并实现的、不属于框架内置的神经网络层。

###### 内置的标准层

- `nn.Linear`：全连接层
- `nn.Conv2d`：卷积层
- `nn.BatchNorm2d`：批归一化层

###### 自定义层模版

```python
# 在 PyTorch 中，所有层（包括自定义层）都必须继承自 nn.Module。
import torch
import torch.nn as nn

class MyCustomLayer(nn.Module):
    def __init__(self, ...):
        super().__init__()
        # 在这里定义子模块或可学习参数
        self.weight = nn.Parameter(torch.randn(...))
        self.bias = nn.Parameter(torch.zeros(...))
        # 或者使用内置层组合
        self.linear = nn.Linear(in_features, out_features)

    '''定义输入层到输出层的计算逻辑'''
    def forward(self, x):
        # 定义前向传播逻辑
        return X - X.mean()
```

##### 10：加载、存储权重向量和模型

- 对于单个张量，我们可以直接调用`load`和`save`函数分别读写它们

  - ```python
    y = torch.zeros(4)
    torch.save([x, y],'x-files')
    x2, y2 = torch.load('x-files')
    (x2, y2)
    ```

- 内置函数来保存和加载整个网络

  - ```python
    class MLP(nn.Module):
        def __init__(self):
            super().__init__()
            self.hidden = nn.Linear(20, 256)
            self.output = nn.Linear(256, 10)
    
        def forward(self, x):
            return self.output(F.relu(self.hidden(x)))
    
    net = MLP()
    X = torch.randn(size=(2, 20))
    Y = net(X)
    # 将模型的参数存储在一个叫做“mlp.params”的文件中
    torch.save(net.state_dict(), 'mlp.params')
    # 实例化原始的多层感知机，加载文件中的模型采纳数
    clone = MLP()
    clone.load_state_dict(torch.load('mlp.params'))
    clone.eval()
    ```
