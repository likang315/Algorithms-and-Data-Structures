##### 01：机器学习（Machine Learning, ML）

- 让计算机**自动学习大量数据，发现其中的规律或模式**，并基于这些规律做出预测或决策。
  - **从经验中学习**，是一种浅层学习。

##### 02：机器学习的基本流程

- 数据收集：获取与问题相关的数据。
- **数据预处理**：清洗数据、处理缺失值、特征处理、特征转换等。
  - 特征处理一般都需要人工干预完成，利用人类的经验来选取好的特征，并最终提高机器学习系统的性能。因此，很多的机器学习问题变成了**特征工程（Feature Engineering）**问题。

- **模型选择**：根据任务选择算法（如线性回归、神经网络等）。
- **模型训练**：用数据调整模型参数，使其拟合数据规律。
- **模型评估**：用测试数据验证模型的泛化能力。

##### 03：表示学习（Representation Learning）

- 将输入信息转换为有效的特征，或者称为**表示（Representation）**。
- 表示学习：如果有一种算法可以自动地学习出有效的特征，并提高最终机器学习模型的性能，

###### 颜色的表示

- 局部表示：假设所有颜色的名字构成一个词表 𝒱，词表大小为 |𝒱|．我们可以用一个|𝒱|维的one-hot向量来表示每一种颜色．在第𝑖 种颜色对应的one-hot向量中，第𝑖 维的值为1，其他都为0。

- 分布式表示：用RGB值来表示颜色，不同颜色对应到R、G、B三维空间中一个点。

  - **分布式表示很容易表示新的颜色，也可以计算不同颜色之间的相似度。**

- | 颜色   | 局部表示      | 分布式表示         |
  | ------ | ------------- | ------------------ |
  | 琥珀色 | [1, 0, 0, 0]  | [1.00, 0.75, 0.00] |
  | 天蓝色 | [0, 1, 0, 0]  | [0.00, 0.5, 1.00]  |
  | 中国红 | [0, 0, 1, 0]  | [0.67, 0.22, 0.12] |
  | 咖啡色 | [0, 0, 0, 1]T | [0.44, 0.31 0.22]  |

##### 04：深度学习

- 传统机器学习需要**人工设计特征**（如图像的边缘、颜色），而深度学习能**直接从原始数据（如图像像素、文本字符）中自动学习高层次特征**。

###### 贡献度分配问题（Credit Assignment Problem，CAP）

- 即一个系统中不同的组件（component）或其参数对最终系统输出结果的贡献或影响。

###### 端到端学习

- 指在学习过程中不进行分模块或分阶段训练，直接优化任务的总体目标。

##### 05：拟合

- **字面意思**：模型“贴合”数据，就像用一条曲线去“贴合”散点图中的点。

- 定义：模型通过训练数据**学习数据中的规律（特征与目标之间的关系），从而能够对新数据进行预测**。

- | **概念**     | **训练集表现** | **测试集表现** | **原因**               |
  | ------------ | -------------- | -------------- | ---------------------- |
  | **良好拟合** | 好             | 好             | 模型复杂度与数据匹配   |
  | **欠拟合**   | 差             | 差             | 模型过于简单           |
  | **过拟合**   | 极好           | 差             | 模型过于复杂或数据不足 |

##### 06：机器学习分类

###### 监督学习（Supervised Learning）

- 在监督学习中，**训练集包含输入（特征）和对应的输出（标签）**，模型的目标是学习从输入到输出的映射关系，从而能够对新数据做出预测。

###### 无监督学习（Unsupervised Learning）

- 在无监督学习中，**训练集中没有标签**，模型的任务是**发现数据中的隐藏结构或模式**，而不是预测某个特定输出。

##### 07：监督学习任务

###### 回归（regression）

- 试图找到输入（X）和输出（y）之间的数学关系（函数），从而对未知数据进行数值预测。
- 回归含义：用自变量（X）解释或预测因变量（Y），**试图找到Y如何“回归”（趋近）到由X决定的规律性关系上**。
  - 回归均值；

###### 分类（classification）

- 根据输入数据（特征）预测一个**离散的类别标签**（即从预定义的类别中选择一个）。
- 二分类；

##### 08：强化学习（Reinforcement Learning）

- 让**智能体（Agent）通过与环境（Environment）的交互学习最优策略（Policy）**，以最大化长期累积奖励（Reward）。
- 与监督学习不同，RL 不依赖有标签的数据，而是**通过“试错”和“反馈”自主学习**，类似于人类或动物通过经验学习行为。

##### 09：评测指标
###### 均方误差（MSE）

- 是所有样本的“**预测误差平方”的平均值**。

$$
  \text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2
$$

###### 准召

|                | 真实为正例   | 真实为负例   |
| -------------- | ------------ | ------------ |
| **预测为正例** | TP（真阳性） | FP（假阳性） |
| **预测为负例** | FN（假阴性） | TN（真阴性） |

- 精确率：预测的结果里有多少是正确的。

$$
  \text{Precision} = \frac{TP}{TP + FP}
$$

- 准确率：所有的结果里，预测对了多少。

$$
  \text{Accuracy} = \frac{TP + TN}{TP + FP + FN + TN}
$$

- 召回率：所有正确的结果里，预测出多少个。

$$
  \text{Recall} = \frac{TP}{TP + FN}
$$

- F1 分数：综合指标，均衡，只有两者都高，F1 才高！

$$
    F1 = 2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}} = 2 \times \frac{0.667 \times 0.8}{0.667 + 0.8} ≈ 0.727
$$

| 指标                    | 问题                           | 口诀               |
| ----------------------- | ------------------------------ | ------------------ |
| **精确率**（Precision） | “你抓的人里，有多少真是坏人？” | 你有多“准”？       |
| **召回率**（Recall）    | “所有坏人里，你抓到了几个？”   | 你有多“全”？       |
| **F1**                  | “你能不能又准又全？”           | 你是否“又准又全”？ |
