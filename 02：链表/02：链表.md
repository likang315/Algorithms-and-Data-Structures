##### 01：判断两个链表是否有交结点

1. 计算两个链表的长度，减去差值
2. 两个栈，出栈判断

```c
public Boolean getIntersectionNode(ListNode h1, ListNode h2) {
	// p1 指向最长的
  ListNode p1,p2;
	int count = 0;
	for (p1 = h1; p1.next != null; count++, p1 = p1.next);     //统计h1的结点总数
	for (p2 = h2; p2.next != null; count--, p2 = p2.next);     //统计好h2结点的总数，求出多余数
	// 判断h1还是h2长
  if(count > 0) {
		p1 = h1;
		p2 = h2;
	} else {
		p1 = h2;
		p2 = h1;
		count = -count;
	}
  // 跑过多余项
	for (; count != 0; p1 = p1.next, count--);            
  // 判断指向是不是同结点
	for (; p1 != p2; p1 = p1.next, p2 = p2.next);        
	if (p1 != null && p2 != null)
		return true;
	else
		return false;
}
```

##### 02：将两个升序链表合并成一个升序表

```java
public ListNode merge(ListNode head1, ListNode head2) {
  ListNode p = head1, q = head2;
  // 创建头结点
  ListNode head = new ListNode(-1);
  ListNode tail = head;
  // 比较大小，合并
  while (p != null && q != null) {
    if (p.val < q.val) {
      tail = tail.next = p;
      p = p.next;
    } else {
      tail = tail.next = q;
      q = q.next;
    }
  }
  if (p != null) {
    tail.next = p;
  }
  if (q != null) {
    tail.next = q;
  }
  
  return head.next;
}
```

##### 03：判断链表是否有环

- 快慢指针

```java
public boolean detectCycle(ListNode head) {
  // 快慢指针
  if (head == null)
    return null;
  ListNode p = head, p2 = head;
  // 终止条件
  while (p2.next != null && p2.next.next != null) {
    p = p.next;
    p2 = p2.next.next;
    if (p == p2) {
      return true;
    }
  }
  return false;
}
```

##### 04：删除倒数第n个结点

- 时间复杂度：O(n)
- 双指针，相对位移

```java
public ListNode removeNode(ListNode head, int n) {
  	// 万一删除头结点
    ListNode p = new ListNode(0);
    p.next = head;
    ListNode first = p;
    ListNode second = p;
  	// 向前移动 n + 1 步
    for (int i = 0; i <= n; i++) {
        first = first.next;
    }
    while (first.next != null) {
        first = first.next;
        second = second.next;
    }
  	// first 和 second不一定相隔一个
    second.next = second.next.next;
    return p.next;
}
```
