##### 01：判断两个链表是否有交结点

1. 计算两个链表的长度，减去差值；
2. 两个栈，出栈判断；

```c
public Boolean getIntersectionNode(ListNode h1, ListNode h2) {
    // p1 指向最长的
    ListNode p1,p2;
    int count = 0;
    for (p1 = h1; p1.next != null; count++, p1 = p1.next);     //统计h1的结点总数
    for (p2 = h2; p2.next != null; count--, p2 = p2.next);     //统计好h2结点的总数，求出多余数
    // 判断h1还是h2长
    if(count > 0) {
        p1 = h1;
        p2 = h2;
    } else {
        p1 = h2;
        p2 = h1;
        count = -count;
    }
    // 跑过多余项
    for (; count != 0; p1 = p1.next, count--);            
    // 判断指向是不是同结点
    for (; p1 != p2; p1 = p1.next, p2 = p2.next);        
    if (p1 != null && p2 != null)
        return true;
    else
        return false;
}
```

##### 02：将两个升序链表合并成一个升序表

```java
public ListNode merge(ListNode head1, ListNode head2) {
    ListNode p = head1, q = head2;
    // 创建头结点
    ListNode head = new ListNode(-1);
    ListNode tail = head;
    // 比较大小，合并
    while (p != null && q != null) {
        if (p.val < q.val) {
            tail = tail.next = p;
            p = p.next;
        } else {
            tail = tail.next = q;
            q = q.next;
        }
    }
    if (p != null) {
        tail.next = p;
    }
    if (q != null) {
        tail.next = q;
    }

    return head.next;
}
```

##### 03：判断链表是否有环

- 快慢指针

```java
public boolean detectCycle(ListNode head) {
    // 快慢指针
    if (head == null)
        return null;
    ListNode p = head, p2 = head;
    // 终止条件
    while (p2.next != null && p2.next.next != null) {
        p = p.next;
        p2 = p2.next.next;
        if (p == p2) {
            return true;
        }
    }
    return false;
}
```

##### 04：删除倒数第n个结点

- 时间复杂度：O(n)
- 双指针，相对位移

```java
public ListNode removeNode(ListNode head, int n) {
    // 万一删除头结点
    ListNode p = new ListNode(0);
    p.next = head;
    ListNode first = p;
    ListNode second = p;
    // 向前移动 n + 1 步
    for (int i = 0; i <= n; i++) {
        first = first.next;
    }
    while (first.next != null) {
        first = first.next;
        second = second.next;
    }

    first = second.next;
    second.next = second.next.next;
    first.next = null;
    return p.next;
}
```

##### 05：反转链表

- 双指针，直接改变引用；

```java
public ListNode reverseList(ListNode head) {
    ListNode p = head, q = null;
    while (Objects.nonNull(p)) {
        ListNode tmp = p.next;
        p.next = q;
        q = p;
        p = tmp;
    }
    return q;
}
```

##### 06：奇偶链表

- 给定一个单链表，把所有的奇数节点和偶数节点分别排在一起，保持奇数节点和偶数节点的相对顺序；
- 不区分奇偶，用顺序判断奇偶；

```java
public ListNode oddEvenList(ListNode head) {
    if (head == null) {
        return head;
    }

    ListNode h2 = head.next;
    ListNode p1 = head, p2 = h2;
    // 奇偶结点都不为空
    while (p2 != null && p2.next != null) {
        p1.next = p2.next;
        p1 = p1.next;
        p2.next = p1.next;
        p2 = p2.next;
    }

    p1.next = h2;
    return head;
}
```

##### 07：链表的中间结点

- 给定一个头结点为 `head` 的非空单链表，返回链表的中间结点，如果有两个中间结点，则返回第二个中间结点。

```java
public ListNode middleNode(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow;
}
```

##### 08：回文链表（对称链表）

- 官方解释时间复杂度O(n）；
- 快慢指针在寻找中间节点的过程中直接反转链表前半部分，找到中间节点之后直接从中间向两边开始比较

```java
public boolean isPalindrome(ListNode head) {
    if (head == null) {
        return true;
    }
    ListNode pre = null;
    ListNode slow = head;
    ListNode fast = head;
    while (fast != null && fast.next != null) {
        ListNode temp = slow.next;
        // 跳过头结点
        if (pre != null) {
            slow.next = pre;
        }
        pre = slow;
        slow = temp;
        fast = fast.next.next;
    }
    // 奇数个结点
    if (fast != null)
        slow = slow.next;
    while (slow != null) {
        if (slow.val != pre.val) 
            return false;
        slow = slow.next;
        pre = pre.next;
    }
    
    return true;
}
```
