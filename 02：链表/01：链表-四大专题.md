##### 链表的分类

1. ###### 单项链表

   - 创建(尾插)，删除，插入，都需要两指针联动
   - 遍历，输出时，选择用 p 还是用 p->next 看终止条件
   - 倒叙遍历：递归，栈，设岗哨

2. ###### 带表头的单项链表

   - 创建，输出，查找，删除，插入  (操作的是下一个结点，所以插入和删除无需两指针联动，只需一个指针)

3. ###### 单项循环链表

   - 创建：建一个挂一个，还是最终挂
   - 遍历，插入，删除，设置尾指针更加高效

4. ###### 双向链表

   - 创建，删除，插入 （只需要一个结点）

// 用栈创建链表
public ListNode creatLink(int[] data) {
    Stack<ListNode> stack = new Stack<>();
    for (int dataNum : data) {
        stack.push(new ListNode(dataNum));
    }
    ListNode head = null;
    while (!stack.isEmpty()) {
        ListNode current = stack.pop();
        current.next = head;
        head = current;
    }

    return head;
}

###### 4：链表的删除 （两指针联动）

1. 只删除一个key，找到就跳出

```java
public ListNode delSingleNode(ListNode head, int key) {
   ListNode p, q = null;
    // key的查找，空循环
    for (p = head; p != null && p.val != key; q = p, p = p.next);
    if (p == null) {
        System.out.println("NOT FOUND!");
    } else {
        // 判断头和中间尾
        if (p != head) {
            q.next = p.next;
        } else {
            head = head.next;
        }
        // 没有引用自动被JVM回收
        p.next = null;
        q = null;
    }
    return head;
}
```

// HashSet
public void delSameNode(ListNode head) {
    ListNode p, q;

    for (ListNode pkey = head; pkey != null; pkey = pkey.next) {
        // 内层循环判断重复删除
        for (q = pkey, p = pkey.next; p != null; q = p, p = p.next) {

###### 5：链表插入 ，两指针联动

1. **头插法**：将所有奇数结点放在偶数结点之前 

```java
public ListNode preLink(ListNode head) {
    if (Objects.isNull(head)) {
        return head;
    }
    ListNode q = head, p = head.next;
    // 不用判断是否为头结点还是中间尾
    while (p != null) {
        if (p.val % 2 != 0) {
            q.next = p.next;
            p.next = head;
            // 挂头链
            head = p;
            p = q.next;
        } else {
            q = p;
            p = p.next;
        }
    }
    return head;
}
```

2. **尾插法**：将链表的值按升序排序
   1. 头插 + 尾插

```java
// 选择排序
public ListNode selectSort(ListNode head) {
    if (Objects.isNull(head)) {
        return head;
    }
    ListNode h1 = null, p, q, pmin, qmin = null, tail = null;
    while (head != null) {
        for (pmin = q = head, p = head.next; p != null; q = p, p = p.next) {
            if (p.val < pmin.val) {
                pmin = p;
                qmin = q;
            }
        }
        // 判断最小值结点是否为首结点
        if (pmin == head) {
            head = head.next;
        } else {
            qmin.next = pmin.next;
        }
        pmin.next = null;
        // 新建节点是否为空
        if (h1 == null) {
            h1 = tail = pmin;
        } else {
            // 挂尾链挪尾指针
            tail.next = pmin;
            tail = pmin;
        }
    }
    return h1;
}
```

##### 03：环形链表（单向循环链表）

​	设尾指针更加高效，创建，输出和删除，插入（和单向循环链表一样）

1. ###### 正向建链，建一个挂一个

```java
public ListNode creatLink(int[] a) {
    ListNode head = null, tail = null, p;
    for (int i = 0; i < a.length; i++) {
        p = new ListNode(a[i]);
        if (head == null) {
            // 创建的循环链表的头结点指头（开始就创建一个环）
            p.next = p;
            head = tail = p;
        } else {
            p.next = head;
            tail = tail.next = p;
        }
    }
    return head;
}
```

    ListNode q = null;

        if (p.val != key) {
            q = p;
            p = p.next;
        } else {
            break;
        }

```java
// 单向循环链表传尾指针比头指针更加高效
public ListNode JosephusRing(ListNode t, int s) {
    ListNode head, t1, p;
    // 最后一个结点时跳出循环
    while (t != t.next) {
        // 删除找前驱
        for (int i = 0; i < s - 1; i++)
            t = t.next;
        p = t.next;
        t.next = p.next;
        p.next = null;
        if (head == null)
            head = t1= p;
        else
            // 挂尾链挪尾指针
            t1 = t1.next = p;      
    }
    // 判断是否只有一个结点
    if (head == null)                   
        head = t;
    else
        t1 = t1->next = t;

    return head;
}
```
